# Kortlek för Kroppen

En interaktiv webbapp med kort för nervsystemreglering. Dra ett kort och få vägledning till övningar som hjälper dig hitta lugn och balans.

## Om appen

Appen innehåller ett antal olika kort med olika tekniker för att reglera nervsystemet, uppdelade i kategorier.

## Användning

1. Öppna `index.html` i en webbläsare
2. Klicka på kortleken för att dra ett kort
3. Följ övningen på kortet
4. Klicka på "Blanda" för att starta om kortleken

## Teknologi

Appen är byggd med vanilla HTML, CSS och JavaScript. Inga externa beroenden krävs.
